Personal Project: Speech-to-Text Translator

Available on my GitHub

This app simulates a translation app where user can speak to their device and the app detects the language spoken and translates it to the desired language. The purpose of this personal object is to learn more about utilizing python libraries such as speech_recognition and googletrans and applying them to develop solutions for real world problems such as language barriers and communication difficulty for those who are deaf or have impaired hearing.

When running this app, the user is first asked to what desired language they want their messages to be translated to. If a valid language is entered, the speech_recognition library will access the user's microphone on their device and extract text from recognized audio. The googletrans library is then used to detect language spoken from the extracted text and will output the message translated into the language based on the user's selected language.

The known languages are stored in a json file in dictionary format with destination keys that are used by the googletrans library to identify and translate languages. Additionally, users can always change the translated language while the app is still running.

If I were to improve on this project, I would like to implement this concept onto an AI chatbot and also have it displayed on a GUI or website for a visual and interactive experience.
